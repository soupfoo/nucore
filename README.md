
# nucore

> A minimal implementation of some *nix coreutils

## Installation

**Build from source:**

Required: Go compiler (>=v1.21)

```
sh build.sh
```

Alternatively, you can run `go install ./cmd/*` to install all the tools to `$HOME/go/bin`.
